## lazydocker如何使用及特点

 ![](https://img.shields.io/github/downloads/jesseduffield/lazydocker/total?style=flat-square)

 安装使用也很简单，可以通过包管理器下载，也可以直接到releases下去获取二进制包

 如果windows可以使用如下：

 ```
 scoop install lazydocker
 #或者
 choco install lazydocker
 ```

 mac用户的话，可以直接使用brew

## 项目简介

polybar是一个C++语言编写的桌面状态栏工具，它可以自定义设置，是专为linux系统设计的。

polybar可以用来显示常用监控信息，如主机状态等，具体可以展示内容可以到下文查看。

```

```

```

```
Arch Linux
```
pacman -S polybar
```

```
dnf install polybar
```

## polybar支持展示的内容

![](image-4.png)

- 可以展示时间及日期按照你想要的方式及位置
- 可以展示键盘布局以及各类指示灯的状态
- 可以展示背光情况
- 可以展示网络连接的情况
- 可以显示主机的负载情况如cpu或者内存
- 可以展示音量情况
- 可以展示基于时间的shell执行以及命令输出情况
- 可以展示进程间的消息传递情况
- 等等
